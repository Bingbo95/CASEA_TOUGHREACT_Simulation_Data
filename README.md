# CASEA_TOUGHREACT_Simulation_Data
This repository shares data from studies using the TOUGHREACT simulator to investigate reservoir characteristics evolution and geochemical reactions in Compressed Air Energy Storage (CAESA). <br>
1. TOUGHREACT is a commercial software (Version V3.32) and cannot be provided herein. <br>
2. Only the model input files (e.g., flow.inp, solute.inp, chemical.inp, all files are zipped in input_files.zip) utilized in the paper are made freely available for academic research. <br>
3. These data are intended for reference purposes, and further discussions should be directed to the authors. <br>
4. When using this data, proper citation is required: "Xu B, Zhang K. Numerical Investigation on Evolution of Reservoir Characteristics and Geochemical Reactions of Compressed Air Energy Storage in Aquifers. Sustainability, Under Review."
